# BookMyStayApp


## Overview

A simple Java project to manage **hotel rooms and their availability** using OOP concepts.

---

## Components

* **Room (Abstract Class)**
  Stores common details like beds, size, and price.

* **DeluxeRoom (Child Class)**
  Inherits from `Room` and defines specific room type.

* **RoomInventory**
  Uses `HashMap` to store and manage room availability.

---

## Features

* Display room details
* Store room availability
* Update room count

---


## Concepts Used

* Abstraction
* Inheritance
* HashMap (Collections)
* Encapsulation
